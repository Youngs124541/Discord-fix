Procedure for bypassing discord blocking (auto-run)

1) Run as administrator "service_discord_youtube"
2) close the "Discord-fix" folder

P.s: To remove auto startup, run "service_remove".

Procedure for bypassing discord blocking (No auto-run)

1) Run as administrator "discord_youtube"
2) Do not close the running console, otherwise the bypass will not work
3) close the "Discord-fix" folder

P.S. You will have to activate this method every time after shutting down the PC.

(Also allows you to use YouTube)
